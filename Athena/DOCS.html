General Syntax


getrawmetatable

Get's the metatable of a userdata or table, even if it's locked.

table getrawmetatable(userdata/table)


getobjects

Loads the content from the asset ID and returns it in a table.

table getobjects(string)


saveinstance

Write's the part's XML data to a file and saves it so you can open it in Roblox studio.

void saveinstance(Instance (part to save), string (file name))


decompile

Decompiles a LocalScript or ModuleScript, and returns its source. *this function is usually broken, dont rely on it. When broken, it will return the script's bytecode as a string instead.

string decompile(Instance (Local/Module Script))


getrenv

Returns Roblox's global enviornment table.

table getrenv()


getreg

Returns Roblox's registry table.

table getreg()


getgenv

Returns Athena's global enviornment. Alias for getrenv in Proto mode.

table getgenv()


geturl

Returns the raw text from that URL.

string geturl(string)


copystring

Copies the string to your clipboard. Aliases: setclipboard, toclipboard

void copystring(string)


getprotomembers

Returns a table of the compiled script's Proto. This works in Wrapper mode too.

table getprotomembers()


gethwid

Returns the user's HWID. Useful for security in scripts. (This doesn't return the HWID Athena's whitelist uses.)

string gethwid()


readfile

Reads and returns a file's contents.

string readfile(string)


writefile

Writes to a file's contents.

void writefile(string)


MouseButton1Up

Releases the left mouse button.

void MouseButton1Up()


MouseButton1Down

Presses the left mouse button.

void MouseButton1Down()


MouseButton2Up

Releases the right mouse button.

void MouseButton2Up()


MouseButton2Down

Presses the right mouse button.

void MouseButton2Down()


MouseButton3Up

Releases the middle mouse button.

void MouseButton3Up()


MouseButton3Down

Presses the middle mouse button.

void MouseButton3Down()


LoadSSUi

Internal function used to load Athena's SS GUI. Useful for developing other external UIs.

void LoadSSUi()






Base64 Syntax


Base64.encode

Encrypts the string into Base64.

string Base64.encode(string)


Base64.decode

Decrypts the Base64 string.

string Base64.decode(string)






Bit Syntax


bit.tobit

Returns the input as a bit.

int bit.tobit(int)


bit.bnot

Returns the negative bit.

int bit.bnot(int)


bit.bor

Does an OR operation on the bit.

int bit.tobit(int, int)


bit.bxor

Does a XOR operation on the bit.

int bit.tobit(int, int)


bit.lshift

Shifts the input left by the amount in the second argument.

int bit.lshift(int, int)


bit.rshift

Shifts the input right by the amount in the second argument.

int bit.rshift(int, int)


bit.ror

Rotates the bit right by the amount in the seond argument.

int bit.ror(int, int)


bit.rol

Rotates the bit left by the amount in the seond argument.

int bit.rol(int, int)


bit.bswap

Swaps the bytes of the bit.

int bit.bswap(int)


bit.tohex

Returns the bits as a hex string.

int bit.tohex(int)






DirectX Syntax


DirectX.start

Starts the DirectX thread for drawing (may cause lag)

void DirectX.start()


DirectX.end

Ends the DirectX thread for drawing.

void DirectX.end()


DirectX.rect

Draws a rectangle on the screen.

void DirectX.rect(int X, int Y, int SX, int SY, bool filled, int R, int G, int B)


DirectX.clear

Clears all DirectX drawing on the screen.

void DirectX.clear()






LuaC Syntax


LuaC.getfield

Gets a key from the stack.

void LuaC.getfield(int index, string key)


LuaC.setfield

Sets a key from the stack.

void LuaC.setfield(int index, string key)


LuaC.pushnumber

Pushes a number onto the stack.

void LuaC.pushnumber(int)


LuaC.pushstring

Pushes a string onto the stack.

void LuaC.pushstring(string)


LuaC.pushboolean

Pushes a boolean onto the stack.

void LuaC.pushboolean(bool)


LuaC.pushnil

Pushes nil onto the stack.

void LuaC.pushnil()


LuaC.pushvalue

Pushes the value of the index to the top of the stack.

void LuaC.pushvalue(int index)


LuaC.settop

Sets the top of the stack.

void LuaC.settop(int)


LuaC.pcall

Calls the stack as a protected function.

void LuaC.pcall(int nargs, int nres, int nerr)


LuaC.type

Gets the luac type of the index on the stack. (Uses Roblox's types, not vanilla.)

int LuaC.type(int index)


LuaC.gettop

Gets the stack's size.

int LuaC.gettop()

Constants


_G.ChathookCMDAlive

If the command chathook is alive or not. [Part of Athena's core, don't touch it.]


_G.ChathookSCRAlive

If the script chathook is alive or not. [Part of Athena's core, don't touch it.]


_HAS_SS

If the game has SS or not.


LUA_GLOBALSINDEX

10002, used for making LuaC.getglobal and LuaC.setglobal.

LUA_REGISTRYINDEX

10000, used for making Debug.getregistry and Debug.setregistry.
